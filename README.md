Простой платформер на arcade

Запуск:

1. Установите зависимости: pip install -r requirements.txt
2. Запустите: python main.py

Особенности:
- Стартовое окно
- Финальное окно
- Подсчёт очков (сбор монет)
- Несколько уровней (2 готовых)
- Камера
- Простая система частиц
- Хранение результатов в CSV/SQLite

Дальше можно добавить анимации, звуки и более сложную игровую логику.
